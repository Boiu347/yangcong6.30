# 家庭包结论速览 V3：编辑部叙事结构改造

## 目标

保留 `familyCoreConclusions` 全部文案与证据，将呈现结构对齐「从小学物理」结论速览 demo 的编辑部叙事节奏，并补齐滚动动效；整体色系相近可区分，避免大块换色割裂。

## 约束

- 不增删、不改写结论正文、要点、原声与业务启发
- Hero 不放主原声；原声只挂在对应要点下
- 去掉左侧研究路径 rail 与统一文档模板
- 自动部署：完成后 push `main` 触发 Railway

## 页面结构

1. Hero：一句话总判断 + 三个事实锚点 + 继续阅读
2. 全篇导航：购买决策 / 机会人群地图两张章节卡，含模组锚点
3. 章节一（购买决策）：M01 成立原因、M02 未成交卡点
4. 章节二（机会人群地图）：M03 小低组、M04 小高组、M05 其他潜力组合
5. 页尾：进入典型家庭故事
6. 研究资料抽屉与阅读进度条保留

## 模组内部

- 模组头：`title` + `conclusion`
- 每个 `point`：标题 + `text`；`keyPoints` 以递进链路/分层列表呈现
- `evidence` 按 `keyPointIndex` 挂在对应要点下
- `actions` 作为模组末业务启发；`evidenceNote` 为脚注

## 色彩

- 全页暖纸底
- 购买决策强调色偏橙，机会人群同色系略深赭
- 章节底仅轻洗色，不做物理 demo 式整章强换色

## 动效

Hero 高亮展开、导航卡错落入场、章节号回弹、模组顶线拉出、链路逐步显现、原声滚动揭示、进度条 scrub；`prefers-reduced-motion` 降级静态。
