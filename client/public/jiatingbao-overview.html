研究背景

基于飞书《家庭包用户调研》11 位家庭的智能纪要与文字记录，按已购 / 未购、年级组合与购买性质分层提炼洞察。家庭包是洋葱面向多孩家庭的 6 年长周期、全科课程方案，因此研究重点放在“为什么愿意一次性买长期权益”“哪些家庭值得重点推”以及“什么顾虑还没有被解决”。

家庭包成立公式

当前孩子马上能用 + 另一个孩子未来能接上 + 分开买 / 未来补课更贵更麻烦 = 家庭包值得买。

其中，“当前孩子马上能用”是第一价值锚点；“另一个孩子未来能接上”负责价格合理化；“未来补课更贵更麻烦”是促成购买的风险账。

转化价值
识别细分用户类型，给销售提供更精准的话术。
顾虑消解
解决 6 年高价格、高周期、高不确定性带来的心理门槛。

关键洞察

成交锚点

主孩需求决定成交

家庭包并非以“家庭”进入心智，而是先从某个孩子的具体学习问题切入；这个问题越紧急，越容易付费。

“主要考虑的是哥哥，妹妹顺带着用。”

价格合理化

次孩权益让高客单成立

另一个孩子现在 / 未来也能用，家庭包就从高价支出变成“值得”的长期课程。

“如果没有妹妹的权益，可能就会买个短期课。”

关键顾虑

最大阻力是长期不确定性

家长最常见的决策方式，是先买当下够用的课包，再观察是否值得升级。

“花这么多钱不一定见到效果吧，有效果再买吧。”

价值放大

看见长期如何发生

家庭包要证明 6 年不是囤课，而是学习规划、AI 定制班、过程反馈和家庭组学情共同构成的长期方案。

“AI 定制班给我们排好课，定好了就听。”

推课逻辑

小低 + 其他

主需求集中在大孩，小低承担未来确定性

小低更多是“摊钱份子”，但 3 年后提前学的确定性较强；当大孩已出现升学问题时，家庭包更容易成立。

小高 + 其他

不浪费、能学到高中，但仍需消解效果顾虑

家庭包是覆盖小初高的完整方案，但“不浪费”和“内容全面”还不足以解决提分效果的不确定性。

强推组合

单孩小高 / 双小高 / 三孩

单孩小高不存在其他学段浪费感；双小高和三孩家庭可将权益倍增解释成更强的长期价值。

特征判断

按底层动机识别潜力用户

小升初窗口、拔尖自驱、大孩经验迁移、资源预置囤课，是当前最清晰的四类转化机制。

四类典型画像

小升初窗口驱动型

5/6/7 年级附近，有同步复习、暑假预习、分班考试等明确场景。

拔尖自驱超前学型

孩子成绩优秀、自律主动，家长不是补差，而是在支持孩子继续往前学。

大孩经验迁移型

老大经历过升学压力、辅导班低效和补课疲惫，家长希望小孩少走弯路。

资源预置囤课型

把家庭包理解成“储备资源”，关注能学到高中、多孩共用和不浪费。

行动框架

1

先问主孩问题

先识别谁的需求最紧急：升学、补弱、复习、提前学，主孩场景决定成交入口。

2

再讲次孩价值

把另一个孩子现在或未来能用讲清楚，让高客单从“贵”变成“值得”。

3

消解 6 年风险

用孩子愿意学的数据、AI 定制班、1V1 规划和线下课对比降低“不学/无效/浪费”的风险。

4

持续证明效果

强化学情报告、家庭组和阶段反馈，让长期价值不是只在购买时宣传。

人群优先级

P0
小低 + 初中
大盘规模大且转化率高，是家庭包最自然的成交组合。
P1
小低 + 高中 / 小高 + 高中 / 小高 + 小高
规模小但转化率高，适合用升学节点和提前学诉求单独承接。
P2
小高 + 初中 / 初中 + 高中 / 小低 + 小高
规模较大但转化率偏低，需要更强的价值证明和顾虑消解。
未购
小高 + 初中 / 初中 + 高中
核心阻力是不确定效果与怕浪费，需要明确试用、阶段反馈和提分证据。

小低 = 1-3 年级；小高 = 4-6 年级。

一句话结论

家庭包需要从单个孩子的真实学习问题切入，用另一个孩子的未来权益合理化高客单，再用风险消解机制证明 6 年不会浪费；销售不能只讲卖点，而要帮助家长算清“现在不准备，未来会更贵、更麻烦、更被动”的风险账。
