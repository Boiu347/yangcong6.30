# 问题定义：如何卖爆“从小学系列”？



【卖爆】在数学上，就是这3个变量一起乘出来的指数级效果，所以问题就简化成：**怎么让这3个变量都被解锁到最大？**

```Plain Text
GMV = 触达人数 × 转化率 × 客单价 
```



**卖爆的三大必要条件**

|必要条件|如何影响公式中的变量|不满足会怎样|
|---|---|---|
|1. **有戏：**品类的需求够大|直接影响「最大触达人数」|品类盘子小，再优化也摸到天花板，卖不爆|
|2. **有差：**竞争差异化优势|「转化率」容易达到瓶颈|和竞品拉不开差距，在存量里内卷，卖不爆|
|3. **有路：**渠道规模化跑通|同时影响以上三个变量|资源投入与GMV不对等，ROI打不正，卖不爆|



# 研究方向

**——围绕「卖爆三大必要条件」，分别开展「五大专项模块」的研究**

- 盘子研究

- 定位研究

- 人群研究

- 渠道研究

- SKU/价格研究



## 每个模块具体在研究啥？



1. **盘子研究 → 框定「触达上限」，确定售卖目标**

- **怎么影响**：算出市场盘子 = X万户 × Y%渗透率 = 触达人数的绝对天花板

- **例子**：若盘子是500万潜在用户，你再努力触达也只能在这个数里滚动，这决定了GMV的物理上限

- **调研方向：**市场规模测算

    

2. **定位研究 → 通过打和竞品的差异化优势，撬动「转化率」**

- **怎么影响转化率**：用户看到内容→"这是给我的 / 解决我的问题"→才会下单

- **例子**：同一个用户，看【快乐学物理】 vs 看【不碰教材的提前学】，下单意愿可能差3倍

- **调研方向：**竞对差异化研究、JTBD

    

3. **人群研究 → 撬动「触达人数的质（精准度）」**

- **怎么影响**：人群定向对了，每1块钱买到的是"对的眼球"；定向错了，1块钱买到的是"无效眼球"

- **例子**：同样花钱找达人合作，面向【5年级焦虑妈妈】vs【泛家长】，前者转化效率可能都高5\-10倍

- **调研方向：**核心盘 \+ 扩圈路径

    

4. **渠道研究 → 撬动「触达人数的量（规模）」**

- **怎么影响**：渠道决定了触达规模的物理上限。一个不适配的渠道，再使劲也只能推到50万；换一个适配的渠道，可能直接冲5000万

- **例子**：从小学系列在洋葱APP直播间（小池子\+错配），日触达1万；换到抖音达人矩阵\+淘客分发（大池子\+适配），日触达可以做到50万

- **调研方向：**分渠道产出策略（在哪，对什么人，用什么话术/手段，卖什么品，能让单渠道的GMV最大化？）

    

5. **SKU/价格研究 → 撬动「转化率」和「客单价」**

- **怎么影响**：单科598 vs 三科套餐1500（买2赠1感），客单价直接从598拉到1500，但转化率可能不同

- **调研方向：**价格测试、营销包装测试

# 研究边界

## 目标人群

从付费人群来看：

- 新用户～30%，路径主要为在新媒体下单，首次下载洋葱学习

- 其余为老用户，主要是APP内下单或电销触达后的转化



目标受众：3\-12岁孩子（学前启蒙 \+ 小学生）的家长

- 课程宣传适合1\-6年级的学生，但是为了“卖爆”，需要扩充更多的机会人群，故同步探索【学前启蒙】用户的需求



因此，本次调研对象主要分为2群人：

- **市场大盘（非洋葱家长）**，也是洋葱的潜在付费新用户

- **洋葱私域**，可能是老未，也可能是组合品/单科的付费用户

**调研方案**：

- **方案1：**直接找外部供应商全包，要求配额（如：大盘家长700个，其中200个用洋葱）

- **方案2：**外部供应商承接【非洋葱用户】的调研需求，洋葱私域用户由我们单独发短信调研（如：非洋葱700个\-供应商，洋葱700个\-自投）





## 竞对界定

> **检索口径：**面向小学阶段，围绕物理/科学启蒙、科学素养、实验探究或小初物理衔接，提供付费课程/课包/App/会员/实验服务的产品
> 
> - 最关键的不是名字里有没有「物理」，而在于是否争夺家长需求：让孩子提前理解物理/科学概念，培养兴趣，或为后续理科学习打基础
> 
> 



**1）第一梯队：市场规模最大**

这些不是最垂直，但用户盘子、品牌认知、投放能力和转化能力最强。

- 学而思科学 / 希望学

- 猿辅导相关科学素养课

- 作业帮相关科学/素养课

- 学习机生态里的科学/物理内容：学而思学习机、科大讯飞、步步高等

这类竞品的特点是：家长信任度高、流量大、售卖能力强。即使课程不完全叫“小学物理”，也会优先抢走用户预算。



**2）第二梯队：垂直品类里规模较大**

这些更接近科学启蒙、物理启蒙、STEM，但整体市场规模通常小于教育大厂。

- ✅妙懂物理

- ✅十分科学/物理十分通

- ~~麦淘实验室~~

- 超级课堂

- 博雅小学堂科学/物理课

- ✅NB小学科学

这类适合作为重点竞品，因为它们更能反映“家长愿不愿意为物理/科学启蒙单独付费”。

# 研究拆解和假设

## **有戏（品类够大）：盘子研究**

### **核心回答：卖爆的天花板在哪？**

> **该模块不直接撬动 GMV 公式里的任何一个变量，但它给整个研究设定"目标"和"节奏"——告诉你"卖爆的天花板在哪、怎么分阶段打、当前 50 万 GMV 距离爆品还差多少倍"。**
> 
> 



**没有这个模块会怎样**：

- 不知道目标盘子，"卖爆"永远是空话

- 不知道竞品已经卷到哪一步，差异化攻防没有锚点

- 不知道老用户的学科偏好，资源分配（拉新 vs 老带）拍脑袋

- 一年后做复盘时，无法判断"50万 → 5000万"是大成功，还是只触达天花板的1%

    

|\#|问题|调研提供什么|
|---|---|---|
|Q\-A1|理科素拓 / 提前学的需求渗透率多少？|问卷渗透率题 → 推算 TAU（整体需求盘）|
|Q\-A2|其中真正付费过 \& 有付费力 \+ 付费意愿的多少？|消费意愿题 → 推算 SAU（可竞争盘）|
|Q\-A3|洋葱认知 \+ 对从小学系列的意愿如何？|洋葱专属题 → 推算 SOU（洋葱机会盘）|
|Q\-A4|三品（物理/生物/地理）需求强度差异多大？|三品对比题 → 推算品类资源配比|
|Q\-A5|"卖爆"的合理量化目标是多少户？|综合所有数据 → 给出三档目标|

## **有差（竞争差异化）：人群 x 定位研究**

### **核心回答：不同人群间，洋葱比竞品的差异化竞争优势是啥？**



在「卖爆」链路里的角色

> **这个模块解锁的是「转化率」—— 回答用户"为什么选你不选竞品"的问题。**
> 
> 没有"差异"的研究，再精准的人群、再大的渠道、再低的价格都无效——因为用户进来后没有一个"必须选你"的理由。
> 
> 



**为什么人群和定位必须交叉做**：

- **❌单做定位：**自说自话（"我们的卖点是X"，但不同家长的关注点可能不一样）

- **❌单做人群：**画像精确但没策略（"知道是谁了，然后呢？"）

- ✅**人群 × 竞争差异化的定位** → 才能给出真正的策略输出： **"对【A人群】，我们要用【优势X】打掉【竞品Y】"**



**最终交付物**

- **一张《人群 × 差异化定位 × 竞品狙击》的三维矩阵图**

- 拿到这张图，业务能：

    - 人群上：判断哪些是最容易转化的「核心盘」？哪些是作为增长引擎的「扩圈主力」？哪些是较难占领的「长尾」？

    - 策略上：对每个目标人群讲对应的差异化故事，不再"一套话术打所有人"。

## 定性

### 目标用户

> 判断是否符合访谈的标准，对竞品和洋葱从小学的：知晓、了解/对比、购买、态度、学习情况等
> 
> 

#### 仅购买“从小学”的用户

> **核心价值：看洋葱为什么赢、价值在哪成立、承诺边界在哪里。**
> 
> 

1. **`P0`****买前比较过竞品，最后买了洋葱：**洋葱和竞品的差异到底在哪里？**（竞争关系）**

    - 最适合提炼【差异化定位】（不是问“洋葱有什么优势”，而是从他们的比较过程里还原）

2. **`P1`****买了满意并持续学，且至少知道或听过竞品****：**洋葱的真实价值在哪里成立？

    - 最适合提炼【核心价值】和【信任证明】

3. **`P2`****买了但没怎么学：**哪些卖点听起来成立，但实际撑不住？

    - 识别承诺边界：找承诺和交付的断层，避免信息屋写过头，影响商品的长期价值（如：口碑/复购/退货等）



#### 仅购买竞品的用户

> **核心价值：看洋葱为什么输、竞品为什么赢、洋葱还有哪些再切入机会**
> 
> 

1. **`P0`****了解/比较过洋葱“从小学”，但最后买了竞品**

    - 竞争关系：最适合提炼 【差异化短板】和【竞品优势心智】（洋葱在哪些环节没有说服他，竞品在哪些点赢了）

2. **`P1`****竞品持续使用/觉得值，且至少知道或听过洋葱“从小学”**

    - 竞品价值：最适合提炼 【用户真实痛点】和【竞品已占领的核心价值】（能告诉我们：竞品到底解决了什么问题）

3. **`P1`**** 竞品不满意/没持续用，且至少知道或听过洋葱“从小学”**

    - 机会缺口：最适合提炼 【洋葱可切入机会】和【竞品承诺断点】（能帮助判断：竞品买前很强的卖点，买后是否真的成立）

4. **`P2`**** 买前完全不知道洋葱**

    - 触达关系：适合补充了解 【渠道触达】和【品类入口】（这类用户对差异化定位帮助相对弱，因为他没有真正比较过洋葱）
